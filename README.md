# StudyCircle — Collaborative Learning Workspace

StudyCircle is a structured group study platform tailored for engineering and degree students in Andhra Pradesh and Telangana (Vijayawada, Guntur, Vizag, Hyderabad, and surrounding colleges). It replaces short-lived, chaotic WhatsApp threads with a highly organized learning environment featuring real-time presence rooms, shared note workspaces, async doubt boards, study session schedules, accountability streaks, and group leaderboards.

## 🛠️ Technology Stack

- **Frontend**: React.js / Next.js (App Router) + Tailwind CSS v4 + Lucide Icons
- **Backend**: Node.js + Express.js
- **Real-time Engine**: Socket.IO (room namespaces & presence updates)
- **Database**: SQLite + Sequelize (Zero-setup local serverless database)
- **Auth**: JSON Web Tokens (JWT) + bcryptjs passwords hashing

## 🚀 Setup & Execution Instructions

### 1. Prerequisites
- Node.js (v18+)
- npm

### 2. Backend Installation & Startup
1. Navigate to `/backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root of the `/backend` directory:
   ```env
   PORT=5000
   JWT_SECRET=super_secret_study_circle_token_2026_key_ap_telangana
   ```
4. Start the development server (runs automatically on port 5000):
   ```bash
   npm run dev
   ```
   *Note: On first startup, a local `database.sqlite` file is created automatically, and tables sync.*

### 3. Frontend Installation & Startup
1. Navigate to `/frontend` directory:
   ```bash
   cd ../frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Next.js development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your web browser.

## 🧪 Running Automated Tests
The backend contains an integration test script that registers users, creates groups, joins them, drafts notes, schedules sessions, logs hours, and checks leaderboards:
1. Ensure the backend server is running in a terminal.
2. In a separate terminal inside `/backend`, run:
   ```bash
   node test-server.js
   ```
   All test blocks should return `✅` and end with `🌟 ALL BACKEND API INTEGRATION TESTS PASSED SUCCESSFULLY! 🌟`.
